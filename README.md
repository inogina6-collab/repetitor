# Домашка.AI

Лендинг **Домашка.AI** — сервиса ИИ-репетитора для школьников и родителей. Сайт помогает разбирать домашние задания по фото или тексту с пошаговыми объяснениями.

## Возможности

- Виджет распознавания задач (демо Vision API)
- Выбор предмета и уровня объяснения (5 класс / 9 класс / ОГЭ·ЕГЭ)
- Блоки для родителей, учеников, отзывов и тарифов
- Модальные окна регистрации и личного кабинета

## Быстрый старт

Откройте `index.html` в браузере — дополнительная сборка не требуется.

```bash
# Windows — двойной клик по index.html
# или через локальный сервер (опционально):
npx serve .
```

## Структура проекта

```
.
├── index.html              # Готовая страница (открывать в браузере)
├── scripts/
│   └── build.py            # Генератор index.html из шаблона
├── gemini-code-*.py        # Исходный скрипт-генератор (legacy)
├── README.md
└── .gitignore
```

## Пересборка страницы

Если вы меняете шаблон в `scripts/build.py`, перегенерируйте сайт:

```bash
python scripts/build.py
```
